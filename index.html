Today's problem

Complete a vowels and consonant function that:

1. prints each vowel first in order of appearance in the word

2. prints each consonant in the same order of appearance in the word

a given string is to be rearranged by two orders. The main order being, vowel letters precede consonants. The second order is to make sure that every letter, whether repeated or not is a part of the new word result

the HTML

contains the id that facilitates a display of the output

the method

  1. created a vowels array containing all the vowels
  2. created two arrays arr and arr2
  3. started a for loop to go through every letter in a given string
  4. checked every letter with the vowels arrays
  5. if letter is in vowel, then push the letter to arr
  6. if the letter is not included in the vowel array, push it to arr2
  7. create new string variable and push a concatenated arr + arr2
  8. create a second for loop that outputs every letter of the new string
